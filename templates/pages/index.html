Introduction

Hello! My name is Patrick Guelcher.

I am obsessed with all things Geography and carried that passion to Virginia Tech where I graduated in 2016 with a Bachelor of Arts in Geography. On my site you can find some more information about me, my current portfolio (a little bare at the moment), and find out how you can contact me.

About this site

I first had the idea of building this website as an opportunity to teach myself some basic web development skills in late 2015. Originally a set of static HTML pages, I re-wrote the entire thing using the Python microframework Flask to run the backend while continuing to expand my Python skills. What you are looking at right now, the frontend, is using Material Design Lite from Google, a web based version of their Android Material Design language.

This site runs on a "droplet", a cloud based VPS, from Digital Ocean running Ubuntu Server. The source code for this website is viewable on my Github account here.
